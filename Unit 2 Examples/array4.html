2D Arrays

A two-dimensional(2D) array is an array of arrays. In other words, it is an array where each element is also an array.

Arrays have rows and columns. Each row is an array. Each element of that array is the column.

Since a 2D array is an array of arrays, it requires 2 indexes. The first index specifies which array. The second index specifies which element of that array.
